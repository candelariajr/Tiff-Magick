This takes a bunch of PDFs and outputs their individual pages into TIFF files. The files are specific to our institution. This can be easily modified with simple programming. 

## NOTE!
Be sure you have installed ImageMagick and it's in your PATH variables.
ALSO Install Ghostscript for ImageMagick as it is needed for large files and optimization purposes


Drop PDFs into input-files directory. 
Run script
Your files are in the output-files directory. 

Works on Windows and Linux. Simple script. Can be easily modified.
